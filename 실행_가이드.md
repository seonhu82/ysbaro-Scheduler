## ✅ 지금 바로 시작하기

### 1단계: 환경 설정

```bash
# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env.local
```

`.env.local` 파일을 열어서 수정:
```env
DATABASE_URL="postgresql://username:password@localhost:5432/dental_scheduler"
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"
```

### 2단계: 데이터베이스 설정

```bash
# Prisma 마이그레이션
npx prisma migrate dev --name init

# Seed 데이터 삽입
npx prisma db seed

# Prisma Studio 실행 (선택사항)
npx prisma studio
```

### 3단계: 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속

**관리자 계정**:
- 이메일: admin@dental.com
- 비밀번호: admin123!

