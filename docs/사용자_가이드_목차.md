## 📕 Chapter 3: 초기 설정 마법사 - 6단계 완전 가이드 (80페이지)
### 3.1 초기 설정 마법사 소개
- 초기 설정의 중요성
- 6단계 개요
- 진행 상황 표시

### 3.2 1단계: 병원 정보 입력
**캡처 화면:**
- 병원 정보 입력 폼
- 각 필드 상세

**설명 내용:**
- 병원명 입력
- 주소 입력
- 연락처 입력
- 사업자 번호 (선택)
- 다음 단계로 진행

### 3.3 2단계: 부서 및 구분 설정
**캡처 화면:**
- 부서 목록 (데스크, 진료실)
- 구분 목록 (팀장, 고년차, 중간년차, 저년차)
- 추가/수정/삭제 버튼

**설명 내용:**
- 부서란? (업무 그룹)
- 구분이란? (경력/직급)
- 기본 부서 및 구분
- 커스텀 부서 추가 방법
- 커스텀 구분 추가 방법
- 순서 조정
- 필수/유연 구분 설정
- 색상 지정

### 3.4 3단계: 원장 정보 및 구분 설정
**캡처 화면:**
- 원장 목록
- 원장 추가 폼
- 원장별 구분 설정

**설명 내용:**
- 원장 정보 입력 (이름, 요일별 근무 패턴)
- 원장 구분 설정 (대표원장, 부원장 등)
- 요일별 근무 설정
- 원장별 색상 지정

### 3.5 4단계: 직원 등록
**캡처 화면:**
- 직원 목록 그리드
- 직원 추가 폼 (개별)
- 엑셀 일괄 등록 버튼
- 템플릿 다운로드

**설명 내용:**
- 직원 개별 등록 방법
  - 이름
  - 생년월일 (YYMMDD) - PIN 번호로 사용
  - 부서 선택
  - 구분 선택
  - 근무 형태 (정규직, 계약직 등)
- 엑셀 일괄 등록
  - 템플릿 다운로드
  - 템플릿 작성 가이드
  - 업로드 및 검증
- 직원 수정/삭제

### 3.6 5단계: 의사 조합 설정
**캡처 화면:**
- 요일별 탭 (월~일)
- 원장 선택 드롭다운
- 필요 인원 설정
- 구분별 필요 인원 (팀장 2명, 고년차 3명 등)
- 전체 조합 목록

**설명 내용:**
- 의사 조합이란?
  - 특정 원장이 근무하는 날의 직원 배치 규칙
  - 예: "김원장 + 팀장 2명 + 고년차 3명"
- 요일별 조합 설정
- 원장 선택
- 총 필요 인원 입력
- 구분별 필요 인원 입력
  - 필수 구분 vs 유연 구분
  - 최소/최대 인원
- 조합 저장 및 검증
- 주의사항
  - 모든 요일에 최소 1개 조합 필요
  - 직원 수가 충분한지 확인

### 3.7 6단계: 휴업일 설정
**캡처 화면:**
- 캘린더 뷰
- 공휴일 목록
- 정기 휴무 설정
- 임시 휴무 추가

**설명 내용:**
- 공휴일 자동 로드
- 정기 휴무 설정 (예: 매주 일요일)
- 임시 휴무일 추가 (연말연시, 특별 휴무)
- 휴업일 삭제
- 휴업일이 스케줄에 미치는 영향

### 3.8 초기 설정 완료 및 다음 단계
- 설정 완료 확인 화면
- 메인 대시보드로 이동
- 다음 단계: 주간 패턴 설정

```
## 직원 관리

### 개요
이 페이지에서는 병원의 모든 직원 정보를 관리할 수 있습니다.

### 기능
1. **직원 목록 조회**
   - 모든 직원의 정보를 테이블 형태로 표시
   - 이름, 부서, 구분, 근무 형태, 재직 상태

2. **검색 및 필터**
   - 검색: 직원 이름으로 검색
   - 필터: 부서별, 구분별, 근무형태별, 재직상태별

3. **직원 추가**
   - "직원 추가" 버튼 클릭
   - 직원 정보 입력
   - 저장

### 단계별 사용법
1. 좌측 메뉴에서 "설정" → "직원 관리" 클릭
2. 직원 목록 확인
3. 검색 또는 필터 사용
4. 직원 정보 수정/삭제 가능
```
