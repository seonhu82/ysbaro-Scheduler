# 연세바로치과 스케줄러 - 용어 사전

**작성일**: 2025-10-22
**목적**: 프로젝트 전체에서 사용하는 용어 통일
**적용 범위**: 코드, 문서, UI, API

---

## 📚 핵심 용어

### 1. 슬롯 (Slot)
**정의**: 특정 날짜 또는 주에 휴무 가능한 직원 수

**계산식**:
```
가용 슬롯 = 총 직원 수 - 필요 배치 인원
```

**예시**:
- 평일: 20명 - 14명 = 6슬롯
- 토요일: 20명 - 6명 = 14슬롯

**영어**: slot, available slot
**데이터베이스**: `availableSlots`, `maxSlots`

---

### 2. 오프 (OFF)
**정의**: 주 단위로 배정되는 의무 휴무일

**특징**:
- 주4일 근무자: 주 2일 오프 (의무)
- 주5일 근무자: 주 1일 오프 (의무)
- 자동 배정 또는 직접 신청 가능

**계산**:
```typescript
주별 오프 목표 = 직원 수 × 오프 일수
- 주4일: 20명 × 2일 = 40슬롯 (정상 주)
- 주4일: 20명 × 1일 = 20슬롯 (공휴일 포함 주)
- 주5일: 20명 × 1일 = 20슬롯
```

**영어**: off, off-day
**데이터베이스**: `OFF` (LeaveType enum)
**UI 표시**: "오프"

---

### 3. 연차 (ANNUAL)
**정의**: 근속 연수에 따라 부여되는 유급 휴가

**특징**:
- 모든 직원에게 부여
- 15일~25일 (근속 연수에 따라)
- 오프와 별개로 사용 가능

**계산**:
```typescript
연차 일수 = {
  1년 미만: 15일
  1~3년: 15일
  3년 이상: 15 + (근속년수 - 1) ÷ 2 (최대 25일)
}
```

**영어**: annual leave, annual
**데이터베이스**: `ANNUAL` (LeaveType enum)
**UI 표시**: "연차"

---

### 4. 직원 (Staff)
**정의**: 치과에서 근무하는 모든 인력 (원장 제외)

**직급**:
- 위생사 (HYGIENIST)
- 어시스턴트 (ASSISTANT)
- 코디네이터 (COORDINATOR)
- 간호조무사 (NURSE)
- 기타 (OTHER)

**영어**: staff, employee
**데이터베이스**: `Staff` 모델
**UI 표시**: "직원"

---

### 5. 원장 (Doctor)
**정의**: 치과 의사

**영어**: doctor
**데이터베이스**: `Doctor` 모델
**UI 표시**: "원장"

---

### 6. 주차 (Week)
**정의**: 월요일부터 토요일까지 6일 단위

**특징**:
- 일요일은 전원 휴무로 별도 관리
- 주차 번호: 1, 2, 3, 4, 5 (월별)

**계산**:
```typescript
주차 범위 = {
  시작: 월요일
  종료: 토요일
  일요일: 제외
}
```

**영어**: week
**데이터베이스**: `WeekInfo` 모델
**UI 표시**: "1주차", "2주차" 등

---

### 7. 신청 기간 (Application Period)
**정의**: 연차/오프 신청을 받는 기간

**특징**:
- 월별로 생성
- 외부 링크 (토큰) 제공
- 상태: ACTIVE, EXPIRED, CLOSED

**영어**: application period, application link
**데이터베이스**: `ApplicationLink` 모델
**UI 표시**: "신청 기간"

---

### 8. 근무 형태 (Work Type)
**정의**: 주 단위 근무 일수 구분

**종류**:
- 주4일 근무 (WEEK_4): 주 4일 근무, 주 2일 오프
- 주5일 근무 (WEEK_5): 주 5일 근무, 주 1일 오프

**영어**: work type
**데이터베이스**: `WorkType` enum, `workType` 필드
**UI 표시**: "주4일", "주5일"

---

### 9. 스케줄 (Schedule)
**정의**: 월별 근무 일정

**상태**:
- DRAFT: 작성 중
- PENDING: 배치 대기
- CONFIRMED: 확정
- DEPLOYED: 배포됨

**영어**: schedule
**데이터베이스**: `Schedule` 모델
**UI 표시**: "스케줄"

---

### 10. 배치 (Assignment)
**정의**: 특정 날짜에 직원을 근무로 배정하는 것

**종류**:
- 직원 배치 (StaffAssignment): 근무 배정
- 오프 배치: 오프 배정
- 연차: 연차 사용

**영어**: assignment
**데이터베이스**: `StaffAssignment`, `LeaveApplication`
**UI 표시**: "배치"

## 🔤 영어 약어 및 줄임말

### API 엔드포인트
```
/api/staff              # 직원
/api/doctors            # 원장
/api/schedule           # 스케줄
/api/leave-apply        # 연차/오프 신청
/api/leave-management   # 연차 관리
/api/auto-assign        # 자동 배치
/api/settings           # 설정
/api/statistics         # 통계
/api/attendance         # 출퇴근 (QR)
```

### 데이터베이스 필드명 규칙
```typescript
// camelCase 사용
staffId          // ✅
staff_id         // ❌

// boolean은 is/has 접두사
isActive         // ✅
active           // ❌

// 날짜는 Date 접미사
createdAt        // ✅ (DateTime)
checkDate        // ✅ (Date만)
create_date      // ❌

// 관계는 복수형
assignments      // ✅
assignment       // ❌
```
